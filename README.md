## A Simple Calculator Using HTML + CSS + JS ( BASIC )

A simple **responsive calculator** built using **HTML, CSS, and JavaScript**.  

## Features

- Simple and clean user interface.
- Supports basic arithmetic operations: `+`, `-`, `*`, `/`.
